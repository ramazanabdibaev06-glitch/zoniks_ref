# Kanal Boshqaruvi Yo'riqnomasi

## Database-dagi Muammolar Hal Qilindi ✅

Quyidagi muammolar tuzatildi:
1. ✅ `@vsyuqvio_bot` o'chirildi (bu bot, kanal emas)
2. ✅ `@yt:@bekwiner` noto'g'ri format o'chirildi
3. ✅ `yt:@bekwiner` to'g'ri formatda qo'shildi

## Telegram Kanallarini Qo'shish

Bot orqali majburiy Telegram kanallarini qo'shish uchun:

### 1. Bot Admin Paneliga Kiring
```
/admin
```

### 2. "🧩 Majburiy kanallar" Tugmasini Bosing

### 3. "➕ Kanal qo'shish" Tugmasini Tanlang

### 4. Kanal Formatlaridan Birini Kiriting:

**Telegram kanallari uchun:**
- `@kanal_username` - Telegram kanal username'i
- `-1001234567890` - Telegram kanal/guruh ID'si
- `https://t.me/kanal_username` - Telegram kanal havolasi

**YouTube kanallari uchun (faqat ko'rsatish, tekshirilmaydi):**
- `yt:@username` - YouTube handle format
- `yt:c:channelname` - YouTube channel format
- `yt:u:username` - YouTube user format

## Misol: To'g'ri Formatlar

### ✅ TO'G'RI:
```
@myuzbekchanneluz          ← Telegram kanal (tekshiriladi)
-1001234567890             ← Telegram supergroup ID (tekshiriladi)
https://t.me/myuzbekchannel ← Telegram kanal (tekshiriladi)
yt:@bekwiner               ← YouTube kanal (faqat ko'rsatiladi)
```

### ❌ NOTO'G'RI:
```
@vsyuqvio_bot              ← Bu bot, kanal emas!
@yt:@bekwiner              ← @ kerak emas
yt:https://youtube.com/... ← Faqat username/handle
```

## Bot ishlashi

1. **Telegram kanallari**: 
   - ✅ Tekshiriladi
   - ✅ Obuna bo'lmagan foydalanuvchilar bloklangan
   
2. **YouTube kanallari**:
   - ⚠️ Faqat ko'rsatiladi (tavsiya)
   - ⚠️ Hech qachon tekshirilmaydi
   - ⚠️ Obuna bo'lmagan foydalanuvchilar bloklanmaydi

## Kanal O'chirish

Admin panelda:
1. "🧩 Majburiy kanallar" → "➖ Kanal o'chirish"
2. Kanalni qo'shganda kiritgan format bilan aynan o'chirish kerak

## Hozirgi Holatni Ko'rish

Admin panelda "🧩 Majburiy kanallar" → "📋 Ro'yxat" tugmasini bosing.
